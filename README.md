# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly) created for week 2 of the Lighthouse Labs Web Development Bootcamp.

## Final Product

A quick and painless interface allows you to enter any valid URL and returns a shortened TinyURL.
![IMAGE](http://i.imgur.com/JoPPJOn.png)

View all your shortened URLs, edit or delete existing URLS or add more to your list.
![IMAGE](http://i.imgur.com/g2Sj0U4.png)


## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
